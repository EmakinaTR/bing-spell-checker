Bing Spell Checker
==================
Python version of spell checker.

Requirements
------------
BING_SPELL_CHECK_API_KEY value must be set as environment value
```
$ export BING_SPELL_CHECK_API_KEY='your_key'
```

Usage
-----
```
$ python spell_checker.py 'It workd!'
```
